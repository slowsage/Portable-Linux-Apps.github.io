# KDIALOG

 Shell scripting with KDE dialogs. Create simple scripts for your users to use.

 This tool is part of the KDE Utils Suite.

 USAGE:
 ```
 kdeutils kdialog [OPTIONS] [ARGUMENTS]
 ```
